## Install

```sh
npm install
```

## Add Note

```
node app.js add --title="some title" --body="some body"
```

## Remove Note

```
node app.js remove --title="some title"
```

## List Notes

```
node app.js list
```

## Read Note

```
node app.js read --title="some title"
```

## Run tests

```sh
npm run test
```
